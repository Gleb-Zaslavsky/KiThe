
## Экспериментальная кинетика: Помощь

## File Manager / Управление файлами
Меню "File Manager" предоставляет инструменты для загрузки, сохранения и управления данными экспериментов.

# Import Data / Импорт данных
Загружает данные из файлов CSV, TXT или Excel. Поддерживаются различные форматы столбцов. После импорта данные отображаются в таблице и на графике.

# New experiment / Новый эксперимент
Открывает диалоговое окно для создания нового эксперимента. Окно состоит из двух панелей: левая — браузер файлов, правая — метаданные.

В браузере файлов можно перемещаться по дискам и папкам, выбирать файл данных. Выбранный файл отображается в поле "File path". В метаданных указываются:
- Идентификатор эксперимента (уникальное имя)
- Скорость нагрева (K/мин) — для неизотермических экспериментов
- Изотермическая температура (K) — для изотермических
- Комментарий (произвольный текст)

Ниже расположены блоки привязки колонок:
- "bind m" — привязка колонки массы: вводится имя колонки и выбирается единица измерения (по умолчанию мВ). Программа будет интерпретировать указанную колонку как массу.
- "bind T" — привязка колонки температуры (единицы: °C, K)
- "bind t" — привязка колонки времени (единицы: с, ч)

После заполнения нажмите кнопку "Select" — данные загрузятся, метаданные сохранятся, привязки вступят в силу.

# Append data to experiment / Добавить данные к эксперименту
Добавляет новые данные к существующему эксперименту, объединяя их в один набор.

# Export Data / Экспорт данных
Сохраняет текущие данные эксперимента в файл выбранного формата (CSV, TXT, Excel).

# Manage Data / Управление данными
Открывает менеджер данных, где можно просматривать, редактировать и удалять загруженные эксперименты.

# Manage Plot / Управление графиком
Открывает диалог настройки графиков: выбор отображаемых кривых, настройка цветов, подписей и других визуальных параметров.

# Save As CSV/TXT/Excel/Image / Сохранить как CSV/TXT/Excel/Изображение
Экспортирует данные в конкретном формате. "Сохранить как изображение" сохраняет текущий график в файл изображения (PNG, JPEG и т.д.) через окно редактора KiThe Plot.

## Math / Математические операции
Меню "Math" содержит инструменты для математической обработки данных: фильтрация, сглаживание, интерполяция.

# Hampel filter / Фильтр Хампеля
Обнаруживает и обрабатывает выбросы в данных. Настраивается размер окна, порог (сигма) и стратегия замены (медианой, NaN, удаление).

Окно фильтра содержит:
- Выбор колонки (col_name) — раскрывающийся список колонок текущего выбранного графика.
- Параметр window (окно) — размер скользящего окна для вычисления медианы и MAD.
- Параметр sigma (сигма) — пороговое значение для определения выбросов.
- Выбор стратегии (ReplaceWithMedian, ReplaceWithNaN, Drop) — как обрабатывать выбросы.
- Поле out_col (необязательное) — имя новой колонки для результата; если не указано, результат заменяет исходную колонку.
- Кнопки "Apply" (применить) и "Close" (закрыть).

# Savitzky Golay / Сглаживание Савицкого-Голея
Применяет полиномиальное сглаживание для устранения шума. Параметры: размер окна, порядок полинома, производная.

Окно сглаживания содержит:
- Выбор колонки (col) — раскрывающийся список колонок.
- Параметр window (окно) — размер окна (нечётное число).
- Параметр poly_order (порядок полинома) — степень полинома.
- Параметр deriv (производная) — порядок производной (0 — сглаживание, 1 — первая производная и т.д.).
- Параметр delta (дельта) — шаг по оси X для вычисления производной.
- Поле out_col (необязательное) — имя новой колонки для результата.
- Кнопки "Apply" и "Close".

# Rolling mean / Скользящее среднее
Вычисляет скользящее среднее по заданному окну. Упрощает данные, уменьшая высокочастотный шум.

Окно содержит:
- Выбор колонки (col_name) — раскрывающийся список колонок.
- Параметр window (окно) — размер окна.
- Поле out_col (необязательное) — имя новой колонки.
- Кнопки "Apply" и "Close".

# Splines / Сплайны
Интерполирует данные с помощью сплайнов (линейных, косинусных, Безье). Позволяет пересчитать данные на новой сетке времени.

Окно содержит:
- Поле new_time_col — имя новой колонки времени (по умолчанию "t_spline").
- Параметр n_points — количество точек новой сетки.
- Выбор вида сплайна (Linear, Cosine, Bezier).
- Для Bezier дополнительный параметр bezier_tension (натяжение).
- Поле out_col (необязательное) — имя колонки для интерполированных значений.
- Кнопки "Apply" и "Close".

# LSQ Splines / LSQ Сплайны
Аппроксимирует данные методом наименьших квадратов с использованием B-сплайнов. Настраивается степень сплайна, количество точек и внутренних узлов.

Окно содержит:
- Параметр degree — степень B-сплайна.
- Параметр n_points — количество точек новой сетки.
- Параметр n_internal_knots — количество внутренних узлов.
- Поле out_col (необязательное) — имя колонки для результата.
- Кнопки "Apply" и "Close".

# LOWESS / LOWESS
Локально взвешенное сглаживание (LOESS) для непараметрической регрессии. Настраивается доля данных, количество итераций, параметр дельта.

Окно содержит:
- Выбор колонки времени (time_col) — раскрывающийся список колонок, используемых как независимая переменная.
- Список колонок для сглаживания — флажки для каждой колонки (кроме времени). Для каждой выбранной колонки можно указать имя выходной колонки (опционально).
- Параметр fraction — доля данных, используемая в каждом локальном регрессионном окне.
- Параметр iterations — количество итераций робастного сглаживания.
- Параметр delta — параметр, ускоряющий вычисления.
- Флажок parallel — использовать параллельные вычисления.
- Кнопки "Apply" и "Close".

## Kinetic Methods / Кинетические методы
Меню "Kinetic Methods" предоставляет набор методов для анализа кинетики термического разложения.

# Isoconversional / Изоконверсионный анализ
Вычисляет эффективную энергию активации в зависимости от степени превращения. Доступны методы Озавы-Флинна-Уолла (OFW), Киссинджера-Акахиры-Сунозе (KAS), Фридмана и др. Позволяет выбрать эксперименты для анализа (все или только выделенные). Результат сохраняется как новый эксперимент.

Окно анализа содержит:
- Выбор метода (Method) — раскрывающийся список с доступными методами.
- Флажок "Use selected only" — если отмечен, можно выбрать конкретные эксперименты из списка.
- Поле "Output ID" — имя, под которым результат будет сохранён в списке экспериментов.
- Кнопка "Calculate" — запуск расчёта.
- Поле "Status" — отображает ход выполнения или ошибки.

# Kissinger / Метод Киссинджера
Определяет энергию активации по пикам на кривых ДТГ. Использует зависимость температуры пика от скорости нагрева. Результаты отображаются в окне (Ea, R²).

Окно содержит:
- Поля отображения результатов: Ea (кДж/моль) и R².
- Поле "Kinetic expression" для ввода выражения (в разработке).
- Кнопки "Calculate kinetic expression" и "Calculate pre-exponential factor" (в разработке).
- Кнопка "Calculate" — запуск расчёта.
- Поле "System messages" — статус выполнения.

# Combined Kinetics Analysis / Комбинированный кинетический анализ
Совместно подбирает параметры модели (порядок реакции n, m) и энергию активации, минимизируя невязку по всем экспериментам. Позволяет задать диапазоны поиска n и m, количество шагов, диапазон степени превращения α.

Окно содержит:
- Параметры поиска: n_min, n_max, n_steps; m_min, m_max, m_steps; alpha_min, alpha_max, refinement_steps.
- Поля отображения результатов: n, m, Ea (кДж/моль), R².
- Кнопка "Calculate" — запуск анализа.
- Поле "System messages" — статус.

# Criado Master Curve / Кривая Мастера Криадо
Строит мастер-кривую для проверки соответствия данных определённой модели функции f(α). (В разработке)

# Fit Model / Подбор модели
Интерактивный подбор кинетической модели к экспериментальным данным. (В разработке)

# Is this a sublimation? / Это сублимация?
Анализирует данные на соответствие модели сублимации (нулевой порядок). Выдаёт вердикт с оценкой достоверности (Yes/Maybe/No) и средние значения Ea, k.

Окно содержит:
- Поля отображения результатов: Mean Ea (кДж/моль), Mean k, Verdict (вердикт).
- Кнопка "Calculate" — запуск анализа.
- Поле "System messages" — статус.

# Golden Pipeline / Золотой конвейер
Автоматизированный pipeline для последовательного применения нескольких кинетических методов с генерацией отчёта.

## Direct Problem / Прямая задача
Меню "Direct Problem" решает прямые кинетические задачи: моделирование кривых по заданным параметрам.

# Solve IVP / Решить IVP
Решает начальную задачу (IVP) для системы обыкновенных дифференциальных уравнений, описывающих кинетику. Позволяет получить теоретические кривые массы, скорости и т.д. (В разработке)

# Solve BVP / Решить BVP
Решает краевую задачу (BVP) для более сложных моделей, например, диффузионных процессов. (В разработке)

## Test Options / Тестовые опции
Меню "Test Options" предназначено для тестирования и отладки программы.

# Run Tests / Запустить тесты
Выполняет встроенные модульные тесты для проверки корректности работы.

# Load testing data / Загрузить тестовые данные
Загружает предопределённые тестовые данные для демонстрации возможностей.

# Create synthetic data / Создать синтетические данные
Генерирует искусственные данные TGA с заданными параметрами: режим (изотермический/неизотермический), кинетическая модель, шум, выбросы. Полезно для проверки методов. Окно настройки позволяет задать количество точек, шаг по времени, начальную температуру, скорости нагрева, параметры модели (m0, k0, E, R), а также добавить шум (гауссовский, равномерный, дрейф) и выбросы (спайки).

# Show and redact settings / Показать и редактировать настройки
Открывает окно настроек программы: калибровочная линия (коэффициенты k, b), количество точек на графике, уровень логирования. Изменения сохраняются в файл tgasettings.json.

# Show Logs / Показать логи
Отображает журнал событий программы (логи) для диагностики.

# Help / Помощь
Показывает это справочное руководство.

## Quick Action Panel / Панель быстрых действий
Панель быстрых действий содержит кнопки для часто используемых операций над выбранными данными.

# Manage Plots / Управление графиками
Открывает диалог управления графиками (аналогично пункту меню). В диалоге можно создавать новые графики, выбирая эксперимент, колонку X и колонку Y. Для каждого графика можно задать уникальную конфигурацию. Кнопки "Add Plot" добавляют новые конфигурации, "Create All" создаёт все заданные графики.

# Column Manager / Менеджер колонок
Открывает диалог управления колонками данных. Позволяет переименовывать, удалять, создавать новые колонки в данных. Также можно изменять тип данных и единицы измерения.

# Golden Pipeline / Золотой конвейер
Быстрый доступ к золотому конвейеру.

# Clear Selected / Очистить выделение
Снимает выделение с графика.

# Reset View / Сбросить вид
Восстанавливает исходные границы графика.

# Refresh / Обновить
Принудительно перерисовывает график.

# Cut before time / Обрезать до времени
Удаляет данные до указанного времени (в поле "Input value").

# Cut selected / Обрезать выделенное
Обрезает данные по выделенному диапазону по оси X или Y (предварительно выбрать X/Y кнопкой).

# Zoom To Selection / Приблизить к выделению
Масштабирует график к выделенной области.

# Move time to zero / Сдвинуть время к нулю
Вычитает из временной колонки минимальное время, чтобы время начиналось с нуля.

# Delete Plot / Удалить график
Удаляет выбранную кривую с графика.

# Calculate relative mass / Рассчитать относительную массу
Пересчитывает массу в относительную (долю от начальной). Требует указания начальной массы в поле "Input value".

# Calculate conversion / Рассчитать степень превращения
Вычисляет степень превращения α (от 0 до 1). Требует указания начальной и конечной массы.

# From mV to mg / Из мВ в мг
Калибрует массу по напряжению (mV) с использованием калибровочных коэффициентов. Создаёт новую колонку с массой в мг.

# From s to h / Из с в ч
Преобразует время из секунд в часы.

# From C° to K / Из °C в K
Преобразует температуру из градусов Цельсия в Кельвины.

# Average on column / Усреднить по колонке
Вычисляет среднее значение выбранной колонки по всему диапазону или по выделенному интервалу.

# Conversion rate / Скорость превращения
Вычисляет производную степени превращения по времени (dα/dt).

# Dimensionless mass rate / Безразмерная скорость массы
Вычисляет производную относительной массы по времени.

# Mass rate / Скорость массы
Вычисляет производную массы по времени (dm/dt).

# T rate / Скорость температуры
Вычисляет производную температуры по времени (dT/dt).

# Arithmetic operations / Арифметические операции
Кнопки "Add", "Sub", "Mul", "Divide", "ln", "exp" применяют соответствующую математическую операцию к выбранной колонке с использованием значения из поля "Input value". Предварительно нужно выбрать колонку (X или Y) и при необходимости отметить "apply
